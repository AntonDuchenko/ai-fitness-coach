## 2. Тема и токены

**Stitch (`get_project`):** dark, `customColor` / primary связаны с **#2563EB**, акцент вторичный зелёный, шрифты **Space Grotesk** (headline) + **Inter** (body), скругления **ROUND_EIGHT**.

**Live:** приложение на Material 3 токенах (`m3-primary`, `m3-surface`, и т.д.), тёмная тема и синий primary визуально в одной семье с макетом; точного совпадения с namedColors Stitch нет — приемлемо для продукта, не для пиксель-вёрстки.

## 3. Структура и контент

- **Stitch HTML (логика макета):** вертикальный мобильный экран: верхняя панель, скроллируемый тред, композер, быстрые действия.  
- **Live:** тот же каркас: `ChatMobileHeader` → скролл сообщений → `ChatComposer`. Боковое меню спрятано в drawer (`Open menu`).  
- **Расхождение:** макет иллюстрирует один конкретный сценарий (макросы); live показывает реальные данные пользователя — для сравнения структуры это нормально, для pixelmatch — нет.
